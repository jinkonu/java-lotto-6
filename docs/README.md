# 요구 사항
1. 들여쓰기 2까지만 허용한다. 
    - if문 안에 while문이 있는 정도까지만.
2. 함수 길이가 15라인 이내가 되도록 한다.
3. else, switch를 쓰지 않는다.
4. Enum을 활용한다.
5. 도메인 로직에 단위 테스트를 적용한다.
6. Lotto 도메인 내에 수정을 가하지 않는다.
